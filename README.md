# laba4

Репозиторий содержит пять простых консольных приложений на Java для базовой практики работы с регулярными выражениями (Regex).

## Состав проекта

### 1. NumberFinder.java
Программа для поиска и извлечения всех чисел из переданного текста. Регулярное выражение распознает как целые, так и дробные числа.

**Запуск**:
```bash
javac NumberFinder.java
java NumberFinder
```
### 2. PasswordValidator.java
Программа для проверки надежности пароля. Проверяет, что длина пароля составляет от 8 до 16 символов, и он содержит только буквы и цифры, при этом обязательно наличие хотя бы одной заглавной буквы и одной цифры.

**Запуск**:
```bash
javac PasswordValidator.java
java PasswordValidator
```
### 3. CamelCaseFinder.java
Программа находит в строке стыки регистра "строчная-Заглавная" (свойственные формату camelCase) и выделяет этот переход восклицательными знаками с обеих сторон (например, переделывает aA в !aA!).

**Запуск**:
```bash
javac CamelCaseFinder.java
java CamelCaseFinder
```
### 4. IPAddressValidator.java
Программа валидатор IPv4-адресов. Проверяет, что переданная строка строго соответствует формату четырех чисел от 0 до 255, разделенных точками.

**Запуск**:
```bash
javac IPAddressValidator.java
java IPAddressValidator```
### 5. WordSearcher.java
Программа для поиска в тексте всех слов, начинающихся с определенной буквы. Поиск не зависит от регистра (учитываются как строчные, так и заглавные первые буквы).

**Запуск**:
```bash
javac WordSearcher.java
java WordSearcher
```
